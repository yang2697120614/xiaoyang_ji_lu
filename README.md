# 药物用量日历

一个简单实用的药物用量日历应用，帮助用户记录和查看每日药物服用量。

## 功能特点

- 直观的日历视图展示每日药物用量
- 支持设置两种交替使用的药物剂量
- 智能循环计算系统自动生成未来用药计划
- 响应式设计，完美适配手机和桌面设备
- 本地数据存储，设置自动保存

## 版本说明

### V1 (动态版本) - index.html
- 完整功能版本，包含所有交互功能
- 使用JavaScript动态生成日历和计算用量
- 支持用户自定义设置两种用量和开始日期
- 本地存储用户设置

### 静态版本 - static-version.html
- 无JavaScript的静态网页版本
- 预先计算了2025年11月至2026年1月的药物用量
- 默认用量设置：用量A=1.5片，用量B=1.75片
- 通过锚点链接实现月份导航

## 部署说明

### Netlify部署指南

1. **准备工作**
   - 确保项目根目录下有index.html文件（主入口）
   - 检查所有资源路径是否正确（使用相对路径）

2. **部署步骤**
   - 将代码推送到GitHub仓库
   - 在Netlify上连接该仓库
   - 配置构建命令：无需构建命令（留空）
   - 配置发布目录：/ (根目录)
   - 点击部署按钮

3. **自定义域名（可选）**
   - 在Netlify控制台中设置自定义域名
   - 配置DNS解析指向Netlify提供的域名

## 本地运行

1. 克隆或下载本项目
2. 直接在浏览器中打开index.html文件
3. 无需安装任何依赖
